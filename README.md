# Ultimate App Downloader

The Ultimate App Downloader is a Windows batch-based installer designed to quickly set up a collection of essential development tools using trusted, official sources. It provides a simple, automated way to prepare a fresh system for coding, scripting, or general productivity.

## Features

- Installs widely used development tools
- Uses official winget package sources
- Fully transparent installation steps
- No hidden actions or background processes
- Clean, readable batch script that can be inspected by anyone
- Beginner-friendly and safe to run on any modern Windows system

## Included Software

This installer sets up a curated set of commonly used tools:

- **Git** – version control system  
- **Visual Studio Code** – lightweight code editor  
- **Python 3** – programming language and runtime  
- **Node.js** – JavaScript runtime  
- **7-Zip** – file archiver  

All software is installed directly from official, trusted repositories via winget.

## Requirements

- Windows 10 or later  
- Winget (included with the Microsoft Store "App Installer" package)

If winget is missing, the script will notify you and provide instructions.

## Transparency & Safety

This project is **not** a virus, malware, spyware, or harmful software.  
Everything the installer does is visible directly in the `.bat` file, which is plain text and fully readable. No system files are modified, and no personal data is collected.

If you do not trust the installer, you are encouraged to open the script and review it before running.

## Purpose

This project exists to simplify the setup process on new or freshly reset Windows machines. Instead of manually downloading each tool, this installer automates the process in a single step.
